About The Project

Pedestrian crossing detection is a safety feature that identifies cross walks and pedestrian crossing using data from cameras. By recognizing this, system can announce a message to the driver for stopping there by enhancing pedestrian safety.

Pedestrian detection aims to improve safety by identifying pedestrians on or near the roadway. Using advanced object detection algorithms, the system:

- Continuously scans the driving environment for pedestrians.

- Alerts the driver to take necessary actions to avoid accidents.
